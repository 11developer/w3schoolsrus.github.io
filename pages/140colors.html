140 стандартных (безопасных) веб-цветов

Как применять классы с названиями стандартных 140 цветов?

Для удобства и быстроты написания кода для стилизации веб-страниц можно использовать названия цветов в качестве классов, подключив таблицу стилей 140 цветов к нужной веб-странице:

Например, вам нужно всего лишь поменять цвет фона определённого элемента на странице. Но чтобы не писать строчный стиль типа <h1 style="background:blue">Фон синего цвета</h1>, можно просто указать класс с названием цвета: <h1 class="blue">Фон синего цвета</h1>. При этом шрифт на тёмных фонах будет иметь белый цвет, а на светлых фонах - чёрный цвет. Также для большей выразительности текста на фоне в некоторых классах применяется тень для текста, которую при необходимости вы также можете отредактировать или убрать совсем в самом файле 140colors.css.

Скачать таблицу с классами 140 стандартных веб-цветов можно по данной ссылке: 140 стандартных веб-цветов

Примечание: Обратите внимание, что названия цветов можно писать по-разному:

  • С использованием заглавных букв, когда название начинается с заглавной: AliceBlue;
  • С использованием только строчных букв: aliceblue;
  • С использованием только строчных букв, но разделяя двойные названия через дефис: alice-blue.

При использовании класса hover (при наведении мыши) используется название строчными буквами через дефис: hover-alice-blue.
