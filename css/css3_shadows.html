CSS Эффекты Теней

С помощью CSS вы можете добавить тень к тексту и элементам.

В этой главе вы узнаете о следующих свойствах:

  • text-shadow
  • box-shadow

CSS Text Shadow - Тень текста

CSS свойство text-shadow применяет тень к тексту.

В простейшем случае вы указываете только горизонтальную тень (2 пикселя) и вертикальную тень (2 пикселя):

Эффект тени текста!

Пример

h1 {
  text-shadow: 2px 2px;
}
Попробуйте сами »

Затем добавьте цвет тени:

Эффект тени текста!

Пример

h1 {
  text-shadow: 2px 2px red;
}
Попробуйте сами »

Затем добавьте эффект размытия в тень:

Эффект тени текста!

Example

h1 {
  text-shadow: 2px 2px 5px red;
}
Попробуйте сами »

В следующем примере показан белый текст с черной тенью:

Эффект тени текста!

Пример

h1 {
  color: white;
  text-shadow: 2px 2px 4px #000000;
}
Попробуйте сами »

В следующем примере показана красная неоновая тень свечения:

Эффект тени текста!

Пример

h1 {
  text-shadow: 0 0 3px #FF0000;
}
Попробуйте сами »

Несколько теней

Чтобы добавить более одной тени к тексту, вы можете добавить список теней через запятую.

В следующем примере показана красная и синяя неоновая тень свечения:

Эффект тени текста!

Пример

h1 {
  text-shadow: 0 0 3px #FF0000, 0 0 5px #0000FF;
}
Попробуйте сами »

В следующем примере показан белый текст с черной, синей и темно-синей тенью:

Эффект тени текста!

Пример

h1 {
  color: white;
  text-shadow: 1px 1px 2px black, 0 0 25px blue, 0 0 5px darkblue;
}
Попробуйте сами »

Вы также можете использовать свойство text-shadow для создания простой рамки вокруг текста (без теней):

Граница вокруг текста!

Пример

h1 {
  color: yellow;
  text-shadow: -1px 0 black, 0 1px black, 1px 0 black, 0 -1px black;
}
Попробуйте сами »

CSS свойство box-shadow - Блочная тень

CSS свойство box-shadow применяет тень к элементам.

В простейшем случае вы указываете только горизонтальную тень и вертикальную тень:

Это желтый элемент <div> с черной тенью блока

Пример

div {
  box-shadow: 10px 10px;
}
Попробуйте сами »

Затем добавьте цвет тени:

Это желтый элемент <div> из серой тенью блока

Пример

div {
  box-shadow: 10px 10px grey;
}
Попробуйте сами »

Затем добавьте эффект размытия к тени:

Это желтый элемент <div> с размытием серой тени блока

Пример

div {
  box-shadow: 10px 10px 5px grey;
}
Попробуйте сами »

Вы также можете добавить тени к псевдоэлементам ::before и ::after, чтобы создать интересный эффект:

Пример

#boxshadow {
  position: relative;
  box-shadow: 1px 2px 4px rgba(0, 0, 0, .5);
  padding: 10px;
  background: white;
}

#boxshadow img {
  width: 100%;
  border: 1px solid #8a4419;
  border-style: inset;
}

#boxshadow::after {
  content: '';
  position: absolute;
  z-index: -1; /* скрыть тень за изображением */
  box-shadow: 0 15px 20px rgba(0, 0, 0, 0.3);
  width: 70%;
  left: 15%; /* половина оставшихся 30% */
  height: 100px;
  bottom: 0;
}
Попробуйте сами »
